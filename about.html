사이트 소개

동물상 연구소는 강아지상과 고양이상 분류 모델을 활용해 사용자에게 재미있는 테스트 경험과 해석 가이드를 함께 제공하는 소규모 정보형 사이트입니다. 단순히 버튼만 있는 도구 페이지가 아니라, 사용 목적, 사용법, 결과 해석, 개인정보 처리 방식, 문의 경로를 분리해 제공하는 것을 운영 원칙으로 삼고 있습니다.

운영 목적

  • 브라우저에서 실행되는 동물상 테스트를 누구나 쉽게 체험할 수 있게 제공
  • Teachable Machine 기반 이미지 분류 모델의 사용 예시를 설명과 함께 공개
  • 결과를 과장하지 않고, 재미용·실험용 도구라는 점을 분명히 안내

콘텐츠 원칙

  • 주제를 동물상 테스트 중심으로 유지하고, 중복 페이지를 만들지 않음
  • 사용자가 실제로 필요로 하는 설명과 안내를 테스트 페이지 외부에 정리
  • 운영 정보, 개인정보 처리, 연락 방법을 숨기지 않고 별도 페이지로 제공

모델과 결과에 대한 안내

현재 테스트는 `dog`, `cat` 두 클래스를 기준으로 분류하며, 결과는 업로드한 사진 또는 웹캠 프레임에서 모델이 더 강하게 감지한 특징을 확률값으로 보여줍니다. 이 결과는 절대적인 판정이 아니라 입력 이미지 조건에 따라 달라질 수 있는 실험용 분류 결과입니다.
