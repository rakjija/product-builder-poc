동물상 연구소는 단순 결과만 보여주는 페이지가 아니라, 테스트 사용법, 모델 동작 방식, 결과 해석 기준, 개인정보 처리 방식까지 함께 제공하는 정보형 사이트입니다. 사용자는 테스트를 실행하기 전 무엇을 기대할 수 있는지, 어떤 한계가 있는지, 어떤 사진이 더 잘 동작하는지 바로 확인할 수 있습니다.

이 사이트가 제공하는 것

  • 웹캠과 사진 업로드를 모두 지원하는 실제 테스트 도구
  • 모델이 어떤 기준으로 반응하는지 설명하는 해석 가이드
  • 문의, 개인정보 처리, 운영 목적이 분명한 보조 페이지

심사 대응을 위해 강화한 부분

  • 주제를 동물상 테스트 중심으로 통일
  • 중복 없는 설명형 콘텐츠와 쉬운 탐색 구조 제공
  • 광고보다 콘텐츠와 서비스 설명이 먼저 보이도록 구성

무엇을 하는 사이트인가

현재 모델은 `dog`, `cat` 두 클래스를 기반으로 얼굴 인상을 분류합니다. 테스트 페이지에서는 분류 확률과 함께 사용자가 이해하기 쉬운 결과 설명을 제공합니다. 홈페이지는 이 도구를 올바르게 사용하기 위한 안내와 주의사항을 정리하는 역할을 맡습니다.

2 분류 클래스
2 입력 방식: 웹캠 / 업로드
4 운영 보조 페이지: 소개 / 개인정보 / 문의 / 404

사용 전에 알면 좋은 점

  • 이 테스트는 재미와 실험 목적의 분류 도구이며 의료, 심리, 신원 판별 용도가 아닙니다.
  • 얼굴이 작거나 조명이 불안정하면 확률이 출렁일 수 있습니다.
  • 정면 사진, 단순한 배경, 눈과 코가 잘 보이는 구도에서 예측이 더 안정적입니다.

모델 결과를 해석하는 법

높은 확률은 "절대적인 정답"이 아니라 현재 사진이나 프레임에서 모델이 더 강하게 감지한 특징을 뜻합니다. 예를 들어 강아지상 결과는 둥근 인상, 밝은 표정, 부드러운 윤곽에 더 반응할 수 있고, 고양이상 결과는 선이 또렷한 인상, 차분한 표정, 눈매 대비에 더 반응할 수 있습니다.

사이트 품질을 위해 추가한 구조

테스트만 놓고 끝나지 않도록 운영 목적, 연락 경로, 개인정보 처리, 404 안내를 별도 페이지로 분리했습니다. 이는 사용자가 필요한 정보를 쉽게 찾도록 돕고, 서비스의 책임 범위를 명확하게 보여줍니다.

테스트 진행 방법

  • 1. `테스트 시작` 페이지로 이동합니다.
  • 2. 웹캠을 켜거나 사진을 업로드합니다.
  • 3. 결과 카드에서 가장 높은 확률과 각 클래스 비율을 확인합니다.
  • 4. 다른 사진이나 표정으로 다시 테스트해 차이를 비교합니다.

누구에게 유용한가

  • 친구들과 가볍게 결과를 비교해 보고 싶은 사용자
  • Teachable Machine 기반 이미지 분류 예제를 보고 싶은 제작자
  • 얼굴 사진 입력형 인터랙티브 콘텐츠 구조를 참고하고 싶은 운영자

자주 묻는 질문

정확도가 항상 같나요?

아닙니다. 같은 사람도 조명, 거리, 표정, 각도에 따라 결과가 달라질 수 있습니다.

사진이 서버에 저장되나요?

정적 사이트 특성상 업로드 이미지와 웹캠 프레임은 브라우저 안에서 처리되며, 별도 업로드 저장 로직은 두지 않았습니다.

왜 소개/정책/문의 페이지가 필요한가요?

테스트 서비스의 목적과 책임 범위, 사용자 문의 경로를 명확히 보여줘야 사이트 신뢰성이 높아집니다.

광고는 어디에 들어가나요?

현재는 사이트 확인용 `AdSense` 설정만 넣은 상태이며, 실제 광고 단위는 콘텐츠 균형을 해치지 않는 범위에서만 추가할 예정입니다.
