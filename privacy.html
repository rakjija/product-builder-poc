동물상 연구소
홈 테스트 사이트 소개 문의하기

개인정보 처리 안내

이 페이지는 동물상 연구소에서 사용하는 입력 데이터와 외부 서비스 범위를 사용자가 쉽게 이해할 수 있도록 설명하기 위한 안내문입니다.

테스트 입력 데이터 처리

  • 웹캠 프레임과 업로드 이미지는 브라우저 안에서 테스트 실행에 사용됩니다.
  • 현재 사이트 자체에는 이미지를 별도 저장하거나 수집하는 서버 로직을 두지 않았습니다.
  • 사용자는 원할 때 웹캠을 중지하거나 다른 이미지로 다시 테스트할 수 있습니다.

외부 서비스 사용

  • `Teachable Machine` 모델 파일을 외부 URL에서 불러옵니다.
  • `Google AdSense` 관련 스크립트와 사이트 확인용 메타 태그를 사용합니다.
  • 문의 폼 전송에는 `Formspree`를 사용하며, 사용자가 직접 입력한 정보만 전송됩니다.

문의 데이터

문의 페이지에서 이름, 이메일, 메시지를 입력하면 해당 정보는 문의 응답을 위해 `Formspree`를 통해 전달됩니다. 문의와 무관한 목적으로 직접 저장하거나 재가공하는 기능은 사이트에 두지 않았습니다.
